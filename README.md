# 🏛️ SIGMA - Gerador de Assinaturas Corporativas

O **SIGMA** é um sistema de software desenvolvido em ambiente acadêmico, mas com aplicação 100% voltada para o mercado real. O projeto nasceu da identificação de uma dor real: empresas que não possuem um sistema centralizado para padronizar e gerar as assinaturas de e-mail dos seus colaboradores. 

A ferramenta foi criada para solucionar esse problema de identidade visual e processos manuais, servindo simultaneamente como o projeto prático de entrega da faculdade.

## 💡 A Dor do Cliente vs. A Solução

* **O Problema Real:** A falta de um sistema automatizado para criação de assinaturas corporativas gerava descentralização, perda de padrão na identidade visual da empresa e tempo desperdiçado configurando manualmente e-mail por e-mail.
* **A Nossa Solução:** Um gerador onde o usuário insere seus dados cadastrais e o sistema renderiza/exporta uma assinatura de e-mail padronizada de forma instantânea, garantindo a governança da marca e agilidade no processo de onboarding de novos colaboradores.

---

## 🚀 Conceitos de Tecnologia Aplicados

O desenvolvimento do sistema permitiu ao grupo aplicar na prática conceitos fundamentais da Engenharia de Software e Ciência da Computação:
* **Automação de Processos:** Transformação de uma rotina manual e descentralizada em um fluxo digital inteligente.
* **Estruturação de Dados:** Organização das variáveis cadastrais (Nome, Cargo, Departamento, Telefone) para geração dinâmica do layout.
* **Trabalho Colaborativo:** Uso avançado de Git e GitHub para gerenciar o desenvolvimento em equipe, combinando os blocos de código de forma organizada.
